# spring boot banner

## 说明
spring boot 在启动的时候会显示“spring”字符的banner,我们可以自定义这个banner.

## 操作方法

* 通过工具创建图像字符``http://patorjk.com/software/taag/``

* 新建文件``src/main/resources/banner.txt``，粘贴上一步创建的字符，到该文件；
